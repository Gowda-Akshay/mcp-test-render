# MCP Server for Render

This is a simple MCP server using Streamable HTTP transport with a test OAuth token endpoint.

## Local run

```bash
npm install
npm start
```

Server defaults:
- `http://localhost:3000/health`
- `http://localhost:3000/oauth/token`
- `http://localhost:3000/mcp`

## Get a bearer token (test flow)

```bash
curl -X POST http://localhost:3000/oauth/token \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "grant_type=client_credentials&client_id=demo-client&client_secret=demo-secret&scope=mcp:tools"
```

## Deploy on Render

1. Push this folder to GitHub.
2. In Render, create a new **Blueprint** and select the repo.
3. Render reads `render.yaml` and creates the web service.
4. Set `PUBLIC_BASE_URL` in Render to your service URL, e.g. `https://mcp-render-server.onrender.com`.

After deploy:
- Token URL: `https://<your-service>.onrender.com/oauth/token`
- MCP URL: `https://<your-service>.onrender.com/mcp`

## Environment variables

- `MCP_BEARER_TOKEN` (generated by Render in blueprint)
- `OAUTH_CLIENT_ID` (default: `demo-client`)
- `OAUTH_CLIENT_SECRET` (generated by Render)
- `PUBLIC_BASE_URL` (set to your Render HTTPS URL)
